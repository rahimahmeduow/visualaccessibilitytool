Visual Accessibility Tool

About

The Visual Accessibility Tool is a web tool that allows people with Visual Impairment to modify elements on a web page so it is easier for them to take in information. The settings can range from changing the colours on a web site to changing the text settings.

Who is it for?

The Visual Accessibility Tool is for web designers and people with visual impairment. It is for Web Designers because if a web designer decides to implement this tool into their websites they have the role of making the different settings visually suitable for the organisation they are designing for while also making it usable for people with different types of visual impairment.

Why is it necessary?

The Visual Accessibility Tool is necessary because current web design have elements that are suitable for most people including some with visual impairment but still leaves out a minority of people who still find it difficult to take in information from a website, the web tool allows web design to be suitable for all people including the minority that needs it.

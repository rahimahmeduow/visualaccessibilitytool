How to use the Visual Accessibility Tool

Logo

The Visual Accessibility Tool has a logo that is intended to be known by people with visual impairment. When a user with visual impairment sees this logo they will know that the website they are on supports the web tool so they can navigate through the website with ease on their eyes.

Prompt

When a user is on a website that supports the Visual Accessibility Tool They will be introduced with this prompt on the bottom left corner of the page, telling them the website they are on supports the Web Tool. Here they can press NO dismiss the prompt, press YES to continue or press LEARN MORE to redirect them to this website, where they can learn more about what the tool is.

Visual Accessibility Tool

After pressing YES on the prompt, the page will reveal the Visual Accessibility Tool on the right of the page. The tool will give the user the ability to change the background and foreground colour, text size and enable high contrast on the page, it will allow the page to be modified while the page still keeps it's intended brand identity and web layout.

The options the tool gives are not labelled because of the limited buttons, it is intended to have an exploration element to it where the user can try out it's features. the first and second buttons allow the user to change the foreground and background colour, the next two allow the user to change the text size and the last one allows the users to increase the contrast on the page. When a setting is enabled it will be highlighted with a bar beside the setting, if the bar is coloured grey it will mean that the website already has a design with high contrast.
